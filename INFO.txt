NOTAS:

Las llamadas al back (api), se hacen en cada componente a través de un useEffect.
donde le indicaremos la ruta del back routes, según la información que queramos recuperar.

En el archivo independiente,
defines funciones que se encargan de hacer las llamadas a la API.
Estas funciones son genéricas y no están ligadas a ningún componente específico.

En el componente, utilizas la función asíncrona definida en el archivo independiente para obtener los datos y actualizar el estado del componente.

El componente se encarga de la lógica de presentación y delega la lógica de obtención de datos a las funciones del archivo independiente.

Backend:
Controladores: Gestionan las rutas y las peticiones, interactúan con la base de datos y devuelven respuestas.
Modelos: Definen la estructura de los datos y las interacciones con la base de datos.
Frontend:
Funciones de API: Realizan las peticiones al backend y gestionan las respuestas.
Componentes: Utilizan las funciones de API para obtener datos y renderizar la interfaz de usuario.
